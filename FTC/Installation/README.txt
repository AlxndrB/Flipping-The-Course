##### Installation guide for E-choice #######

** 1. Install requirements **

	pip3 install -r requirements.txt

** 2. Fill database **
** 2.1 Users **

	Go into python3 shell

	run users.py in FTC/Installation/DB_vullen folder (by running "import users.py")

** 2.2 Modules **

	Go into python3 shell

	run modules_vullen.py in FTC/Installation/DB_vullen folder (by running "import modules_vullen.py")
